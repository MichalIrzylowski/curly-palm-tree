# FACIAL_HAIR options

Pick one value to replace [FACIAL_HAIR or none] in the base prompt.
Use "none" for female characters or clean-shaven males.

| Value | Notes |
|-------|-------|
| none | no facial hair |
| light stubble | subtle, masculine |
| stubble | casual, masculine |
| short beard | neat, professional |
| full beard | pronounced |
| goatee | stylised |
| moustache | classic |
